# 💰 Fintelligent – AI-Powered Financial Intelligence Platform

Fintelligent is an AI-powered financial management platform developed as a Bachelor of Engineering (B.E.) final year project. It helps users monitor expenses, analyze spending habits, detect fraudulent transactions, predict future expenses, and receive personalized financial insights through an intelligent dashboard.

---

## 📌 Project Overview

Managing personal finances can be challenging due to increasing digital transactions and a lack of personalized financial guidance. Fintelligent addresses these challenges by leveraging Artificial Intelligence and Data Analytics to provide users with meaningful financial insights, budgeting assistance, fraud detection, and spending predictions.

The platform combines modern web technologies with machine learning to create a user-friendly financial assistant.

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/mrunalgirase/fintelligent.git
```

### 2. Navigate to the Project

```bash
cd fintelligent
```

### 3. Create a Virtual Environment

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**macOS/Linux**

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the Application

```bash
python app.py
```

The application will start on:

```
http://127.0.0.1:5000
```
